# Evidencia visual

Esta carpeta conserva capturas de referencia generadas por `npm run test:visual`.

- Las capturas existentes no deben borrarse automaticamente.
- La prueba visual se ejecuta manualmente contra un proyecto Supabase de staging.
- GitHub Actions no ejecuta pruebas remotas ni publica estas imagenes como artefactos.
- Antes de aceptar una captura nueva, revisala en movil y escritorio y comprueba que no contenga datos personales.
